# TheTapVerse 🎮

> **Fast games. No download. Just play.**
> Live at [thetapverse.com](https://thetapverse.com)

## ✨ Features

- Single `index.html` file — zero dependencies
- Mobile-first, fully responsive (portrait priority)
- Dark neon theme with smooth animations
- High scores saved via `localStorage`
- Sound effects via Web Audio API (toggle on/off)
- Swipe, tap, drag and keyboard controls
- PWA-ready meta tags for home screen install

---

## 🚀 Run locally

Just open `index.html` in any modern browser — no server needed.

```bash
git clone https://github.com/YOUR_USERNAME/thetapverse.git
cd thetapverse
# open index.html in your browser
```

---

## 🌐 Deploy

Hosted on **Cloudflare Pages** with custom domain `thetapverse.com`.

To deploy your own copy:
1. Fork this repo
2. Go to Cloudflare Pages → Create project → Connect to Git
3. Select this repo — no build command needed
4. Done ✅
